# 파이썬 가상환경

- 시스템 파이썬 보호를 위해 가상환경 사용 필요

### 가상환경 명령어

- 가상환경 생성
  """ shell
  python -m venv myenv # 'myenv'는 가상환경 폴더 이름
  """
- 가상환경 활성화 - macOS/ Linux
  """ shell
  source myenv/bin/activate
  """ - windows
  """ shell
  myenv\Scripts\activate
  """
- 가상환경 종료
  """ shell
  deactivate
  """
